# 前台 UI 设计稿

本目录存放前台页面的 UI 设计参考图，供 Codex 和开发人员在实现前台页面时读取。

## 使用原则

- 涉及前台页面开发时，优先按本目录中的设计图实现页面结构、模块布局和视觉层级。
- 如设计图与项目文档冲突，以 `PROJECT_SUMMARY.md`、`TECHNICAL_PLAN.md`、`ROUTE_PLAN.md` 中已确认的需求边界为准。
- 当前设计图主要用于前台主链路，不代表所有个人协作能力都已进入 MVP。

## 文件对应关系

- `home.png`
  - 前台首页：搜索入口、分类入口、热门商品、资料中心、报价中心、公司库、最新动态。
- `product-list.png`
  - 商品总列表页：多维筛选、排序、报价状态和资料入口。
- `category-navigation.png`
  - 分类导航 / 品牌导航页：目录式浏览和聚合入口。
- `product-detail.png`
  - 商品详情页：规格参数、相关资料、报价记录、关联公司、最近动态。
- `docs-quotes-updates.png`
  - 资料、报价、动态联动场景参考。
- `companies-quotes.png`
  - 公司库、报价总览、平台动态联动场景参考。

## 备注

- 这些图片为静态参考图，不是可运行页面。
- 如后续设计稿继续增加，保持“按页面语义命名”的方式，不再使用 `image1.png` 这类无语义文件名。
