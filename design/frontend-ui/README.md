# 前台 UI 设计稿

本目录存放前台页面的 UI 设计参考图，供 Codex 和开发人员在实现前台页面时读取。

## 使用原则

- 涉及前台页面开发时，优先按本目录中的设计图实现页面结构、模块布局和视觉层级。
- 如设计图与项目文档冲突，以 `docs/PRD.md`、`docs/ARCHITECTURE.md`、`docs/ROUTES.md` 中已确认的需求边界为准。
- 当前设计图主要用于前台主链路，不代表所有个人协作能力都已进入 MVP。
- 前台开发与验收时，同时参考 [UI_SPEC.md](./UI_SPEC.md) 中的布局规则、组件约束、设计令牌和验收规则。

## 文件对应关系

- `home.png`
  - 前台首页：按对象搜索入口、分类入口、热门商品、资料中心、报价中心、公司库、最新动态。
- `product-list.png`
  - 商品总列表页：承接商品浏览、产品检索、分类结果和品牌结果，提供多维筛选、排序、报价状态和资料入口。
- `category-navigation.png`
  - 分类导航页：统一承载产品分类入口和品牌导航入口，点击后跳转商品列表页并回显筛选参数。
- `product-detail.png`
  - 商品详情页：规格参数、相关资料、报价记录、关联公司、最近动态。
- `mobile-design-board-v1.png`
  - 首批移动端设计板：首页、产品列表、分类导航、商品详情的移动端适配参考。
- `docs-quotes-updates.png`
  - 资料、报价、动态联动场景参考。
- `companies-quotes.png`
  - 公司库、报价总览、平台动态联动场景参考。
- `ui-style-guide-board.png`
  - 前台样式规范板：颜色、字体、间距、圆角、阴影、按钮、输入框、标签和状态样式参考。

## 配套文档

- `UI_SPEC.md`
  - 前台主规范，包含通用色彩、字体、间距、圆角、阴影、组件、状态和响应式规则。

## 备注

- 这些图片为静态参考图，不是可运行页面。
- 前台不再单独设计聚合搜索结果页；首页搜索按对象跳转到对应列表页承接结果。
- 前台顶部导航不单列品牌入口；品牌导航归入分类页。
- 如后续设计稿继续增加，保持“按页面语义命名”的方式，不再使用 `image1.png` 这类无语义文件名。
